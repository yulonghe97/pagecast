FOR AGENTS

Built for the model, not the human.

Agents are bad at writing HTML and CSS. They're good at writing structured data. PageCast turns the second kind into the first, and checks every value on the way through.

  • ❯Agents write YAML, not HTML. No invented classes. No missing attributes. The component renders the look; the artifact carries the data.
  • ❯Errors are structured and addressable. Validation returns the source location and what the prop should have been. The model fixes one line and retries.
  • ❯One verb, runs over npx. `npx pagecast export` is a single shell call. Easy to wire into a CI step or an agent turn.
  • ❯No SDK to learn. Any agent that can run a shell command can cast pages. Skills and MCP integrations sit on top of that floor, never under it.
WHAT YOU GET

Bring your own components.

PageCast renders, validates, and exports. You write the components in `.pagecast/`. Remote packs are coming so you can skip that step too.

JSON Schema on every block.

Every prop is checked against a Draft 7 schema before render. Failures point at the source line and the value the agent wrote.

Hydrate only what needs JS.

Static components render once on the server. Interactive ones opt in per manifest and get a hydration island. The rest is zero client JS.

esbuild bundles your components.

esbuild bundles your .tsx files on demand. The dev server watches the artifact and the .pagecast directory; a save triggers a WebSocket reload.

One HTML file. No dependencies.

Export inlines the CSS, embeds any hydration script, and writes a restrictive CSP. The output runs without a server or a network.

CLI

Four commands. That's the whole surface.

npx pagecast export <file> [--out path]
Render an artifact to a single self-contained HTML file. CSS inlined, CSP set.
npx pagecast validate <file> [--json]
Parse the artifact, check every block against its prop schema, exit non-zero on the first failure. Use --json for machine-readable output.
npx pagecast dev <file> [--port n]
Start a live preview server. Watches the artifact and the .pagecast directory; reloads on save.
npx pagecast list-components [--json]
Print every component the registry knows about, with its description, tags, and interactive flag.
